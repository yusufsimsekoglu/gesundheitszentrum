# Gesundheitszentrum Chatbot

🔗 **Live-Demo:** https://gesundheitszentrum-chatbot.onrender.com/

Ein deutschsprachiger Web-Chatbot für ein Gesundheitszentrum.

## Ziel des Projekts

Dieses Projekt unterstützt Benutzerinnen und Benutzer bei klar definierten Anfragen rund um ein Gesundheitszentrum.  
Der Schwerpunkt liegt auf Terminprozessen und allgemeinen Serviceinformationen.

## Funktionen

- Termin buchen
- Termin verschieben
- Termin absagen
- Öffnungszeiten anzeigen
- Adresse anzeigen
- Kontaktinformationen anzeigen

## Technologien

### Backend
- Node.js
- Express
- Socket.io
- dotenv
- axios

### Frontend
- HTML5
- CSS3
- JavaScript
